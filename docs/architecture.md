# 🏗️ Project Architecture

## Overview

This document describes the overall architecture of the PyBe MERN application.

The application follows a MERN-style architecture consisting of:

- **Client:** React + Vite
- **Server:** Node.js + Express
- **Data Storage:** Local JSON files

The architecture will evolve as new features are implemented during development.

---

## High-Level Architecture

```text
Client (React + Vite)
        │
        ▼
Server (Express.js API)
        │
        ▼
Local JSON Storage
```
