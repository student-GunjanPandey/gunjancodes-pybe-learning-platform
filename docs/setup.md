# ⚙️ Local Setup Guide

## Prerequisites

- Node.js (v18 or later)
- npm

---

## Installation

Install project dependencies:

```bash
npm run installAll
```

Configure environment variables:

```bash
cp server/.env.example server/.env
```

Seed sample data:

```bash
npm run seed
```

Run the development server:

```bash
npm run dev
```

---
