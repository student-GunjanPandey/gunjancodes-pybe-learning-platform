# Code of Conduct

We are committed to creating a welcoming and collaborative environment for everyone contributing to this project.

## Our Standards

- Be respectful and professional.
- Communicate clearly and constructively.
- Welcome feedback and different perspectives.
- Support fellow contributors.

## Unacceptable Behavior

- Harassment or discrimination.
- Offensive language.
- Personal attacks.
- Spam or disruptive behavior.

By participating in this project, you agree to follow this Code of Conduct.
